# ETAPA 7 - Projeto de Testes JUnit

Este projeto contém um exemplo de teste unitário JUnit para o sistema do PI.

## Funcionalidade testada
Validação de cadastro de personagem, sem dependência de banco de dados.

## Classe testada
`br.com.pi.service.ValidadorPersonagem`

## Testes implementados
- Deve permitir cadastro quando nome e raça forem válidos
- Não deve permitir cadastro com nome vazio
- Não deve permitir cadastro com nome muito curto
- Não deve permitir cadastro com raça vazia
